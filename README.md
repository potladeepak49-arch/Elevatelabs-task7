##  Project Description

I developed a dynamic User Directory web application that fetches and displays user data from a separate JSON file using the JavaScript Fetch API.

This project demonstrates my understanding of asynchronous JavaScript, API-like data handling, DOM manipulation, and proper error handling in frontend development.

---

##  Objective

The objective of this project was to:

- Fetch data from a separate JSON file
- Parse and display structured user information
- Implement error handling using try-catch
- Create a clean and responsive UI
- Simulate real API behavior using local JSON


##  Key Features Implemented

-  Fetched data using `fetch()` with async/await  
-  Parsed JSON response  
-  Dynamically created and displayed user cards  
-  Displayed Name, Email, and Address  
-  Implemented loading state  
-  Implemented error handling for network failures  
-  Added Reload button to refetch data  
-  Maintained clean, professional UI layout  

---

## Technical Implementation

- Used `async/await` to handle asynchronous operations.
- Verified server response using `response.ok`.
- Used `try...catch` block to handle runtime and network errors.
- Dynamically updated DOM using `createElement()` and `innerHTML`.
- Separated data (JSON) from presentation logic for better scalability.
